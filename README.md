# Social Hub - نظام إعداد الملف الشخصي

## الوصف
نظام متكامل لإعداد الملف الشخصي مع واجهة مستخدم متقدمة وتأثيرات حركية جذابة.

## المميزات
- تصميم عصري وجذاب
- خطوات إعداد تفاعلية
- تأثيرات حركية متقدمة
- دعم كامل للغة العربية
- تجاوب كامل مع جميع الشاشات

## التقنيات المستخدمة
- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts
